About This Project

This project uses advanced analytics to provide detailed player cards for NHL skaters and goalies. These cards feature data visualizations, performance metrics, and salary comparisons, all in an easy-to-use format. Whether you're a fan, analyst, or someone curious about how players stack up, these cards offer valuable insights.
